Task 4. XML & Web-parsing.

6 option: candies.

Application works with xml file contains of candies. For validation that xml
using xsd scheme.

Application can show created table from xml in two different interfaces -
on web page and on runner class.

On web page we can also change language - EN or RU.

For parsing xml uses three different parsers: DOM, SAX, StAX.

Using technologies: Log4j2, JAXB from Java 6 to build entity classes form xml,
TestNG.
