Task 2. Multithreading.

Option 3: Logistic base.

Input file: 3 numbers - common van to load in base, common van to unload in
base and van with perishable goods.

Each van in program is thread. Van arrives in base and take a terminal with
number. Van has a own number.

Base can contain only of ten vans. For this task program uses ExecutorService
with fixed thread pool with size 10.

Code written in Java Code Convention, with plugin "CheckStyle" and "SonarLint".
In code meet out in console using System.out. that contradicts SonarLint rules,
but don't contradicts task permissions.

Using technologies: Java concurrent, Log4j2.
